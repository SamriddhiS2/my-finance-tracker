# Personal Finance Tracker
## Overview
A privacy-focused, offline-first React application to track expenses, income, and family loans across different platforms (Venmo, Chase, Zelle).

## Features
- **Net Worth Calculation:** Automatically calculates "My Money" vs. "Family Funds" (Liabilities)
- **Spending Analytics:** Tracks spending by Week, Month, Year
- **Future Reminders:** A "Note-taking" style interface for upcoming payments
- **Venmo-style Input:** Fast cash entry mechanism
- **Local storage:** Data persists in your browser (no server required)

## Setup & Installation
1. **Clone the repository**
    ```bash
    git clone https://github.com/SamriddhiS2/my-finance-tracker.git
    cd my-finance-tracker
    ```
2. **Install dependencies**
    ```bash
    npm install
    ```
3. **Run the development server**
    ```bash
    npm run dev
    ```
4. **Build for production**
    ```bash
    npm run build
    ```
